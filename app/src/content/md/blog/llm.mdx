# Blog

## Description

The blog pattern presents a curated section of the latest blog articles, highlighting recent content or specific topics for easy discovery. It supports displaying three to four articles, each with a cover image, title, and optional description and metadata such as author and date. This pattern helps users quickly scan and access featured articles, making it suitable for showcasing updates and driving engagement with new content.
