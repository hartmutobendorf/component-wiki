*   used to let the user go to a previous or next page, or a specific numbered page to the relative page.
*   Navigational component
*   mostly used in content sites for something like blog posts, or  tutorials etc.
*   DONT use it for tables, we have a specific pagination just for tables.

  

*   if there is up to 5 pages then all page numbers are shown at the same time
*   if there is more than 5 then page numbers are truncated.
*   *   if truncated the first page and the last page are always shown. then there is elipsis and only the middle 3 are shown in the middle
    *   the middle page is always the one that is currently active
*   if there is only 2 pages only show the the forward and next button
