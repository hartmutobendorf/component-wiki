1.  **Rule** p-rule : The rule component indicates the beginning of a new section.
2.  **Heading** title : The section heading communicates the primary subject or theme of the section.
3.  **Eyebrow** label\_text (optional): The optional muted eyebrow placed above the section title. It provides additional contextual information or categorization for the section when needed.
4.  **Content area**: The content area on the right side of the 50/50 layout where various block types can be placed. Supported blocks include text, images, lists, videos, code snippets, logos, CTAs, and other compatible patterns. The Basic Section layout provides flexible composition while maintaining structural consistency.

Pattern anatomy

### Content blocks

1.  **Subheading** subtitle : The subheading provides additional hierarchy beneath the main title, using H4 or H5.
2.  **Body** description : Standard paragraph-level text used for descriptive or explanatory content.
3.  **Image**: 3:2 or 16:9 ratio image used to illustrate or enhance the section content. It can contain photography, isometric illustrations, diagrams, and similar visuals. The image can also include an optional caption.
4.  **Video**: The linked, embedded video content in 16:9 ratio.
5.  **List** : The divided list supporting up to 10 items, with bullet type options including none, bullet, status, or number.
6.  **Code block** : The pre-formatted code presentation block.
7.  **Logo block** : The logo block showcases a group of related images, such as a group of customer or partner logos.
8.  **Linked logo block** : The linked logo section displays a list of logos which explicitly link to additional pages or resources.
9.  **CTA block** : The CTA block contains one or more call-to-action elements, including a primary button, secondary button, and links.
10.  **Notification** : A prominent information block used to highlight alerts, updates, or contextual messages within the section.
