The basic section is a two-column (50/50) layout pattern for presenting a single topic through a mix of content blocks. It offers a flexible composition—accommodating text, images, videos, lists, code snippets, logos, and call-to-action elements—to support both simple and expressive compositions. The right side’s ability to host one or more modular blocks encourages clear, engaging storytelling without disrupting page rhythm.

Use the basic section to create visually balanced, easily scannable content that adapts to diverse needs. Its versatility makes it suitable for both informational and mixed-media layouts, but avoid overusing block variations or visual density to preserve consistency. Choose this pattern when your message centers on a single topic and can be clearly conveyed in a side-by-side, two-column structure.
