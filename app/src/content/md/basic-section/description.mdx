The basic section is a 50/50 layout pattern designed to present a single topic using various content blocks. It provides a flexible layout system that can accommodate different types of content including text, images, videos, lists, code blocks, logos, and call-to-action elements. Right side of the 50/50 layout can have one or more blocks, allowing to build clear and modular narratives without breaking the overall page rhythm.

The basic section helps create balanced, scannable content that adapts to a wide range of use cases—from simple informational layouts to more expressive, mixed-media compositions. Be mindful that its flexibility can also lead to inconsistency if too many block types or visual densities are combined. Use the basic section when the message fits within a single topic and can be communicated cleanly within a two-column structure.
