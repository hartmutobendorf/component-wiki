Use the Basic section to communicate a single, focused topic supported by a variety of structured content blocks. To promote readability and cohesion, limit each section to seven or fewer content blocks and avoid mixing unrelated messages or too many diverse content types. The pattern excels when the narrative is concise and each element directly supports the main topic.

The Basic section works best when content naturally fits a two-column layout—typically with a prominent title on the left and supporting material on the right. For medium size, both 50/50 and full-width variants are available within the macro, offering flexibility for different content needs.

### Supported UI blocks for the content area

1.  **Subheading**: The subheading provides additional hierarchy beneath the main title, using H4 or H5.
2.  **Body**: Paragraph text for descriptions or explanations.
3.  **Image**: 3:2 or 16:9 ratio image used to illustrate or enhance the section content. It can contain photography, isometric illustrations, diagrams, and similar visuals. The image can also include an optional caption.
4.  **Video**: The linked, embedded video content in 16:9 ratio.
5.  **List**: The divided list supporting up to 10 items, with bullet type options including none, bullet, status, or number.
6.  **Code block**: For displaying pre-formatted code.
7.  **Logo block**: Showcases related logos, e.g., customers or partners.
8.  **Linked logo block**: Displays logos that link to additional resources.
9.  **CTA block**: The CTA block contains one or more call-to-action elements, including a primary button, secondary button, and links.
10.  **Notification**: A prominent information block used to highlight alerts, updates, or contextual messages within the section.

### When to use

*   You need to present one clear topic with supporting explanatory content.
*   You need a flexible pattern that works for both marketing and informational content.
*   The 50/50 layout helps balance text and media for better readability.

### When not to use

*   When multiple topics or groups require clearer separation, or the content needs to be quickly scanned like a list
*   *   In these cases, consider Tiered list, Equal heights, or Divided section instead.
*   If only a single primary content block is needed, use a specific pattern for that purpose (e.g., Logo section, Linked logo section, or Resources).
*   When a more immersive or highlight-oriented layout is appropriate.
*   *   In that case, use Rich vertical list or highlight patterns like Text highlight or Data highlight.
